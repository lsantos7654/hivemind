# Hivemind

Central management for Claude Code expert agents.

## Structure

- `experts/<name>/` - Expert definitions and knowledge docs (source of truth)
- `agents/` - GENERATED by `hivemind sync` (do not edit directly)
- `commands/` - Slash commands for Claude Code
- `hivemind_cli/` - Python CLI package

## Key Commands

```bash
hivemind init      # Set up ~/.claude symlinks + sync
hivemind list      # Show all experts
hivemind sync      # Regenerate agents/ from enabled experts
hivemind status    # Dashboard overview
hivemind add <url> # Add a new repo expert
hivemind fetch <n> # Clone a repo
hivemind enable <n>  # Enable expert
hivemind disable <n> # Disable expert
```

## Development

```bash
pip install -e .   # Install CLI in development mode
```

## How It Works

1. Expert definitions live in `experts/<name>/agent.md` with `{{HIVEMIND_ROOT}}` placeholders
2. `hivemind sync` resolves placeholders to absolute paths and writes to `agents/`
3. `hivemind init` symlinks `agents/` and `commands/` into `~/.claude/`
4. Claude Code discovers agents from `~/.claude/agents/`
